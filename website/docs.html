What is supported?

Language support

Language support is complete for MATLAB, and mostly working for Octave. The following language versions are fully supported:
  • MATLAB 2017b
  • MATLAB 2018a, 2018b
  • MATLAB 2019a, 2019b
  • MATLAB 2020a, 2020b
  • MATLAB 2021a
Octave support is somewhat limited right now, but at least Octave # comments and in-line functions are supported. Full support is planned.

Operating system support

MISS_HIT is a Python3 program and does not require a working MATLAB or Octave environment. It should work on any operating system that can run Python3.
