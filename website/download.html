Download MISS_HIT

Python Package Index

The easiest way to install it is to use the latest release published on PyPI: 
$ pip3 install miss_hit
There are two relevant package:
  • miss-hit (the user-visible tools)
  • miss-hit-core (the internals)
MISS_HIT has no dependencies, other than Python ≥ 3.6

GitHub Release

You can also download the latest release (which combines both miss-hit and miss-hit-core into one package) on GitHub:
  • MISS_HT Releases

Source Code

You can directly clone the GitHub Repository: 
$ git clone https://github.com/florianschanda/miss_hit.git
